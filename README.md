# testcontainers

Библиотека для управления контейнерами Docker в тестовом окружении.

```bsl
#Использовать testcontainers;

	КонтейнерHttpBin = Новый ОбщийКонтейнер(Образ)
		.ОткрытьПорт(80);

    КонтейнерHttpBin.Запустить();

    ЛокальныйПорт = КонтейнерHttpBin.ПолучитьОткрытыйПорт(80);
    Сообщить("Контейнер запущен на локальном порту: " + ЛокальныйПорт);

    КонтейнерHttpBin.Остановить();
```
